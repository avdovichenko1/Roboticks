ls - позволяет просмотреть содержимое каталога
cd - можно перейти в другую директорию (cd <путь к директории>)
mkdir - создает директорию (mkdir test  --- создает папку с названием «test» там, где вы находитесь)
touch - создает пустой файл (touch app.log - файл с названием app.log)
nano - позволяет редактировать файл (nano test_env.json, Ctrl+O — чтобы сохранить, Ctrl+X — для выхода)
cp - скопировать файл (cp что_копировать куда_копировать)
mv - переместить файл (аналогично cp)
rm - удалить файл (rm* - удалить все файлы в директории)
history — показать последние 1000 команд
man - узнать как работает команда (man ls)
